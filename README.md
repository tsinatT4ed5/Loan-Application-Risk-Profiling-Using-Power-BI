💳 Loan Application Risk Profiling Using Business Intelligence (Power BI)

📌 Project Overview

This project presents a Business Intelligence (BI) solution for loan application risk profiling using Power BI. The system transforms raw loan data into interactive dashboards to help financial institutions make faster, data-driven, and transparent lending decisions.

The project replaces manual loan approval processes with an automated risk classification system that identifies Low, Medium, and High-risk applicants.

🎯 Objectives
Automate loan risk classification using BI logic
Analyze loan approval trends and risk distribution
Identify key factors influencing loan default risk
Provide interactive dashboards for decision-makers
🛠 Tools & Technologies
Power BI Desktop
Power Query (ETL)
Microsoft Excel (Dataset)
DAX (Data Analysis Expressions)
📊 Dashboard Features
KPI Cards (Approval Rate, Total Applications, High-Risk Count)
Risk classification (Low / Medium / High)
Interactive filters (Education, Property Area, Credit History)
Decomposition Tree for root cause analysis
Scatter plot: Income vs Loan Amount
📈 Key Insights
614 total loan applications analyzed
89 applicants classified as high risk
69% overall approval rate
Credit History is the strongest predictor of loan risk
🧠 Business Impact
Reduces manual decision-making bias
Improves loan approval speed and consistency
Helps banks minimize non-performing loans
Enhances transparency in credit decisions
